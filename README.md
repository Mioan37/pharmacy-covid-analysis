# Pharmacy Operations During COVID-19

A portfolio project that models how a small local pharmacy in Samos could have experienced demand shifts, stock pressure, and operational changes during the COVID-19 period.

## Project purpose
This project was created to demonstrate:
- healthcare-oriented data analysis
- trend analysis across OTC and COVID-related categories
- stock pressure and out-of-stock monitoring
- business storytelling for Power BI / GitHub portfolios

## Important disclaimer
This project uses synthetic and anonymized data created for portfolio purposes.

All figures in the dataset, including sales, stock levels, service volumes, and revenue amounts, are indicative only. They do not represent real pharmacy financial records or actual personal/business data.

The model is designed to reflect realistic pharmacy demand patterns during the COVID-19 period while protecting sensitive business information.

## Dataset scope
The workbook and CSV include weekly records from 2019 to 2022 for selected categories such as:
- cough syrups
- lozenges
- vitamins and supplements
- masks
- antiseptics
- gloves
- COVID self tests
- pharmacy COVID test service

This is not intended to represent the full turnover of a pharmacy. It focuses on selected OTC / COVID-related categories and services for portfolio demonstration.

## Files

- `pharmacy_covid_dataset_realistic.csv` – raw dataset

## Suggested Power BI pages
1. Sales trend before vs during COVID
2. Revenue by category
3. Out-of-stock incidents by year/category
4. Supplier delay trend
5. Demand spike analysis

## Example portfolio insights
- Protective products and testing-related categories show a clear demand spike after early 2020.
- Vitamins and supplements maintain elevated revenue through the main COVID period.
- Supplier delays increase during high-demand periods.
- Out-of-stock incidents become more visible in fast-moving COVID-related items.
